Nginx

Установка сервера Nginx:

sudo apt-get install nginx

В папке /etc/nginx/sites-available/ создаём конфигурационный файл example_config и записываем в него:

server {
	listen 80;
	listen [::]:80;

	server_name example.com;

	root /var/www/example.com;
	index index.html;

	location / {
		try_files $uri $uri/ =404;
	}
}

Сохраняем симлинк в /etc/nginx/sites-enabled:

ln -s /etc/nginx/sites-available/example_config /etc/nginx/sites-enabled/example_config

Перезапуск сервера Nginx:

sudo /etc/init.d/nginx restart
