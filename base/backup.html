Архивирование конкретной папки

Например:

tar cvf - /var/www | gzip -9c > /home/user/backup/server.tar.gz
# либо (тут со списком директорий)
tar -czf /home/user/backup/server.tar.gz -P /var/www /etc/ssmtp /etc/apache2 /etc/php5

Или, если необходимо сделать бэкап системной папки:

sudo tar cvf - /etc/apache2 | gzip -9c > /var/www/backup/apache2.tar.gz

# Первый путь до папки - это то, что сохраняем в архив, второй - путь к файлу, куда сохраняем.

Работа с архивами .zip

# если не установлено, то устанавливаем

sudo apt-get install zip
sudo apt-get install unzip

Создание архива:

zip -r /our_way/name_dir.zip /our_way/name_dir

Распаковка архива в текущую директорию:

cd /our_way
unzip name_dir.zip

# либо указываем полный путь, что и куда
