Настройка энергосбережения на лэптопе

Программы

sudo add-apt-repository ppa:webupd8team/unstable 
sudo apt-get update 
sudo apt-get install laptop-mode-tools

Запуск графического интерфейса Laptop Mode Tools

gksu lmt-config-gui
