Проблемы с Dropbox

Отображение иконки Dropbox в системном трее

1 вариант. Создать исполняемый файл и добавить его в автозагрузку:

#!bin/bash

dropbox stop && dbus-launch dropbox start

2 вариант. В файле /usr/bin/dropbox внести изменения:

# создать новую строчку после "import os"

os.environ['DBUS_SESSION_BUS_ADDRESS'] = ""

Затем:

dropbox stop
dropbox start

Информация

# http://askubuntu.com/questions/732816/xubuntu-dropbox-icon-fail

Установка пдлагина Dropbox для контекстного меню

sudo add-apt-repository ppa:xubuntu-dev/extras

sudo apt-get update

sudo apt-get install thunar-dropbox-plugin
