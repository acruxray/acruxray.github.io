Работа с MySQL

Полное удаление MySQL и установка заново:

sudo apt-get --purge remove phpmyadmin
sudo apt-get --purge remove mysql-server-5.5
sudo apt-get --purge remove mysql-common
sudo apt-get autoremove
sudo reboot
sudo apt-get install mysql-common
sudo apt-get install mysql-server-5.5
sudo service mysql start
sudo apt-get install phpmyadmin
# вместо mysql-server-5.5 использовать свою версию или просто mysql-server

Исправление ошибок:

# Если имеем ошибку: The MySQL server is running with the --secure-file-priv option so it cannot execute this statement.
# Для mySQL 5.7 и выше -> в файле /etc/mysql/mysql.conf.d/mysqld.cnf добавляем ниже [mysqld] строчку:
secure-file-priv = ""
# и рестартим mySQL:
sudo /etc/init.d/mysql restart
